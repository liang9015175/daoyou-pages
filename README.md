# Daoyou - AI Coach for Dota2

> Your Dota2 Match Explained | Auto-generated Dota2 Story & Insights

A comprehensive Dota2 match analysis platform powered by AI, providing personalized coaching, match insights, and community features.

## Features

### Core Features
- Match Statistics & Details
- Hero & Item Database
- Player & Match Search
- Friend System & Live Match Tracking

### AI-Powered Analysis
- Win/Loss Analysis
- Hypothetical Scenarios
- Build Recommendations
- Team Fight Positioning
- Performance Analysis
- Vision Analysis
- Pro Player Comparison

### Community (UGC)
- Forum & Discussion
- Video Sharing
- Memes & Creative Content
- Coaching Tutorials
- Pro Player Insights

### Premium Features
- AI Analysis Reports
- Hero Specialization Cards
- Real-time Notifications
- Exclusive Tutorials & Videos

## Local Development

```bash
# Install dependencies (if using http-server for local dev)
npm install

# Serve the site locally
npm run serve
```

## Deployment

This is a static site designed for GitHub Pages. Simply push to the `main` branch and enable GitHub Pages in repository settings.

## Tech Stack

- HTML5 / CSS3 / JavaScript (ES6+)
- Tailwind CSS (via CDN)
- GSAP (Animations)
- Custom Particle System
- i18n Support (English / Chinese)
